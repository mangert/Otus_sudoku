# Sudoku Console App

Интерактивное консольное приложение для решения судоку.

## Возможности

- Поддержка размеров 4×4, 9×9, 16×16
- Загрузка из файла (текстовый формат)
- Ввод с клавиатуры
- Сохранение результата в файл

## Пример файла (9×9)

5 3 0 0 7 0 0 0 0
6 0 0 1 9 5 0 0 0
0 9 8 0 0 0 0 6 0
8 0 0 0 6 0 0 0 3
4 0 0 8 0 3 0 0 1
7 0 0 0 2 0 0 0 6
0 6 0 0 0 0 2 8 0
0 0 0 4 1 9 0 0 5
0 0 0 0 8 0 0 7 9

## Управление

- Выбор пункта меню — цифры
- Выход — `0`
